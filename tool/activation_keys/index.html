激活密钥集锦

常用软件激活方式和密钥

Windows 10

  1. 以管理员身份运行命令提示符。

  2. 卸载当前产品密钥

    slmgr.vbs /upk

    （弹出“已成功卸载了产品密钥”）

  3. 安装新的产品密钥

    slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX

    （弹出：“成功的安装了产品密钥”）

  4. 设置密钥管理服务计算机名

    slmgr /skms zh.us.to

    （弹出：“密钥管理服务计算机名成功的设置为 zh.us.to”）

  5. 激活

    slmgr /ato

    （弹出：“成功的激活了产品”）

  6. WIN+R 查看激活状态

    slmgr.vbs -xpr
