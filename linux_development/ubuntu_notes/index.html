Ubuntu使用笔记

官方网站: https://cn.ubuntu.com/download/desktop 官方网站: https://ubuntu.com/download/desktop Ubuntu 18.04 阿里云镜像: http://mirrors.aliyun.com/ubuntu-releases/18.04/ Ubuntu 20.04 网易镜像: http://mirrors.163.com/ubuntu-releases/20.04/

vmware 16.2.0版本Ubuntu18.04虚拟机 安装了vmware tool之后依然无法从主机复制粘贴文件到虚拟机的问题

sudo  apt-get autoremove  open-vm-tools
sudo apt-get install open-vm-tools-desktop
sudo reboot

替换国内源(清华或者阿里) 阿里云: http://mirrors.aliyun.com/ubuntu 清华: http://mirrors.tuna.tsinghua.edu.cn/ubuntu https://mirrors.tuna.tsinghua.edu.cn/help/ubuntu/

sudo apt-get update 更新源 sudo apt-get install package 安装包 sudo apt-get remove package 删除包 sudo apt-cache search package 搜索软件包 sudo apt-cache show package 获取包的相关信息，如说明、大小、版本等 sudo apt-get install package --reinstall 重新安装包 sudo apt-get -f install 修复安装 sudo apt-get remove package --purge 删除包，包括配置文件等 sudo apt-get build-dep package 安装相关的编译环境 sudo apt-get upgrade 更新已安装的包 sudo apt-get dist-upgrade 升级系统 sudo apt-cache depends package 了解使用该包依赖那些包 sudo apt-cache rdepends package 查看该包被哪些包依赖 sudo apt-get source package 下载该包的源代码 sudo apt-get clean && sudo apt-get autoclean 清理无用的包 sudo apt-get check 检查是否有损坏的依赖

更改时区: 东八区，中国上海 sudo tzselect 选择地区并确认 sudo cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime sudo ntpdate time.windows.com 更新时间 sudo hwclock --systohc //修改硬件CMOS的时间，非常重要，如果没有这一步的话，后面时间还是不准

开启ssh登录 sudo apt-get install openssh-server

默认的Ubuntu存储库包含一个名为build-essential的元包，它包含GCC编译器以及编译软件所需的许多库和其他实用程序。 sudo apt install build-essential 该命令将安装一堆新包，包括gcc，g ++和make等。

查看端口是否被占用(53为端口号) netstat -anp | grep 53 根据端口查找进程 sudo lsof -i:端口号 杀掉进程 sudo kill PID号

文件管理器里进入地址栏直接编译: ctrl + L

终端进入上次cd的目录: cd -

创建桌面快捷方式: https://blog.csdn.net/u012258905/article/details/117938295 https://blog.csdn.net/gymaisyl/article/details/86573247

查看系统中是否有libpng12.so.0 执行命令： ls -ld $(locate -r libpng..so.)

ubuntu18.04安装中文输入法

获取线程ID: pthread_self()和gettid

#include <sys/syscall.h>
#define gettid() syscall(SYS_gettid)

使用ps -T 进程ID 查看线程列表
