默认程序

使用默认程序设置默认用户体验。

1.注册应用程序 查找可执行文件：想要ShellExecuteEx能够找到应用程序，可以在HKEY_LOCAL_MACHINE \ Software \ Microsoft \ Windows \ CurrentVersion \ App Paths下注册安装的应用程序。例如 App Paths ThunderPdf.exe (Default) = D:\VS Projects\Win32Project\Debug\ThunderPdf.exe path D:\VS Projects\Win32Project\Debug

注册程序标识符ProgID 通过在HKEY_CLASSES_ROOT \ Applications \ ApplicationName.exe子项下包含注册表项，应用程序可以提供下表中显示的特定于应用程序的信息。 HKEY_CLASSES_ROOT Applications wordpad.exe FriendlyAppName = @%SystemRoot%\System32\shell32.dll,-22069 SupportedTypes .3gp2 DefaultIcon (Default) = %SystemRoot%\system32\wmploc.dll,-730 shell open command (Default) = "D:\VS Projects\Win32Project\Debug\ThunderPdf.exe" "%1"

计算机\HKEY_LOCAL_MACHINE\SOFTWARE\RegisteredApplications下添加子项"ThunderPdf"，值为"Software\ThunderPdf\Capabilities" 对应的在"计算机\HKEY_LOCAL_MACHINE"创建"Software\ThunderPdf\Capabilities"例如 ThunderPdf Capabilities ApplicationDescription 迅雷PDF是一款超好用的PDF阅读器。 ApplicationIcon "D:\VS Projects\Win32Project\x64\Debug\ThunderPdf.exe",0 ApplicationName 迅雷PDF

FileAssociations .pdf ThunderPdf.pdf

2.文件的打开方式的“推荐程序”的注册表地址 计算机\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\FileExts.pdf\OpenWithList 使用IApplicationAssociationRegistrationUI接口设置默认程序

3.通知系统新的关联选择可用 SHChangeNotify(SHCNE_ASSOCCHANGED, SHCNF_DWORD | SHCNF_FLUSH, nullptr, nullptr);
