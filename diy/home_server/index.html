自建服务器配置

不考虑专业级的机柜和服务器主板，个人使用买不起也没有这么高的需求。

家用自建服务器的配置视需求选择。 1. 树莓派等小主板 - 适合自建博客和 DIY 开发学习。 2. 旧的家用主机配置例如9代Intel处理器和搭配的主板 - 便宜且性能够用，做不了大的视频直播等用途，但一般的建站和 NAS 完全是性能过剩。 3. 二手洋货例如 Xeon E3 等过时的服务器CPU等 - 搭配 ECC 内存要比上面的更稳定，但是二手有风险。 4. 国产替代品例如永擎和超微的产品 - 正品还是有些贵，一个主板几千块。
