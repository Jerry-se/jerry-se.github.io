装机

卖掉游戏笔记本后，我决定组装一台自己的台式机。

笔记本

2017-02-10 在京东购买了 联想(Lenovo)拯救者R720 15.6寸游戏笔记本电脑(i7-7700HQ 8G 1T GTX1050 2G IPS 黑) 实付5999元，12期分期挺便宜。后来又加装了128G固态硬盘和8G内存条。

2022-08-20 京东-爱回收 2040 回收成交

然后在纠结是否再买一台新笔记本后，我决定自己组装一个台式机。

CPU

一开始想选12700KF 搭配 华硕的Z690 玩超频，后来想想，我买来最大的用途还是看视频玩游戏，偶尔写写代码，玩下3A游戏，没有必要超频，费电费钱还折腾人。还有就是重点单核高频，不考虑AMD。最后还是决定12700 搭配 B600M。如果再省些，i5-12600K其实也可以，而且比12700要便宜400块，京东货源充足。

主板

CPU确定12700后，主板就是华硕重炮手或者微星迫击炮二选一了，玩灯就选华硕，我其实不在乎，而且微星的接口好像更友好，WIFI最好还是带上。然后就是我不太想用大机箱，为了兼容性，还是选择 m-atx 主板比较好。

显卡

因为矿难来临，市场上矿卡翻新卡太多，铭瑄或者七彩虹的3060也要2500以上，可能京东旗舰店买到的也是翻新卡，所以最后决定先用核显撑着，等4060显卡。

京东 12700 + 微星 B660M MORTAR WIFI DDR4 板U套装 3599 * 9.7折 = 3491.03 但是下单前盯一个多星期，12700带核显版本一直缺货，等不及了在淘宝intel旗舰店买的板U套装，原价3699，店铺60券，天猫红包3元，到手价3636。

内存

  1. 首先DDR5太贵，性能比DDR4提升不大，而且还有很多问题，所以不考虑DDR5。
  2. 频率选择。因为12700锁电压，频率太高可能无法点亮。而且性价比上DDR4 3200最优。
  3. 品牌。内存超频首选英睿达，镁光原厂颗粒。最次或者没钱就用国产的光威，合肥长鑫的颗粒。好看不差钱就选芝奇皇家戟，金士顿销量高假货多不推荐，其他威刚、海盗船都差不多。

机械硬盘

机械硬盘看个人需求，没有大容量存储需求的不需要买。习惯用希捷，性价比高。其他东芝、西数均可。看重使用时长，稳定不易坏。

电源

首先计算瓦数，一般i7 + 3060显卡需要650W，准备蹲守40系显卡后又增加了100W功耗。然后品牌一开始选择了华硕 TUF 650W 铜牌突击手。那个时候也不太懂金银铜牌，全模组那些东西。后来看了一些B站的视频，特别是超详细！电源选购万能攻略，一站式电源基础知识入门指南！内含电源瓦数推荐，我把电源换成了海盗船 RM750 金牌全模组，然后在下单前发现涨价了，最后选了安钛克 HCG 750W 金牌全模组。

机箱

本人不喜欢太大的机箱，租的房间小，不能占用太大空间。刚开始的想装itx型，发现机箱和小硬件太贵，主板限制太大，扩展接口少，硬件温度高，烤鸡温度CPU动不动90'，显卡温度也不低，于是转向m-atx机箱。B站阿肯老师评测的机械大师非常好，搭配多，可玩性高，能装的硬件也多，可惜有点贵，接近千元。然后看上了华硕冰立方ap201，可惜当时黑色版缺货，而且防尘性不好。然后就选择了乔斯伯D30，最高240水冷也足够了。

散热

水冷一定考虑机箱的大小，品牌的话，利民、九州风神、ID-COOLING和酷冷至尊都差不多，预算高可以选超频三。机箱如果没有送风扇，看需求可能还要额外买，M-ATX机箱都要注意风扇的尺寸。如果有RGB需求，肯定会比没有灯的贵，这方面就是个人爱好了。

显示器

显示器主要品牌非常多，常见的AOC、DELL、华硕、三星和菲利普，小米也有显示其产品了。

我的标准要求：

  1. 首先分屏用多了，27寸刚刚好，搭配2K分辨率，一边开浏览器，一边写代码非常舒服。24寸太小，1080P分屏无体验，也不需要4K，不仅贵还需要更好的显卡来支持。
  2. 偶尔玩下游戏看看电影，本人手残，基本不玩fps射击类游戏，刷新不需要太高，75 144 165均可，不需要240Hz。中值144Hz即可。
  3. 对曲面屏无感。
  4. 不是PS和设计类工作，对专业的色彩精度和校色没有要求。HDR无所谓，又不是高清视频从业者。
  5. 必须要能升降，左右和旋转无所谓。
  6. HDMI、DP有就性，VGA这种旧接口无所谓。

综合以上，泰坦军团 27英寸 2K 144Hz IPS 可升降旋转最具性价比。

装系统

首次安装系统需要准备一个8G以上的U盘(不推荐光盘方式)，首先就是选择要安装的系统版本，下载好镜像。如果使用微软官方的制作工具，该工具会自动下载镜像。如需手动下载，下载网站推荐使用微软官方的下载入口，觉得麻烦就使用 MSDN，这是一个个人性质的原版软件信息收录站点，提供BT资源下载，win11和后面的新系统又放到了新版的MSDN网站，请自行斟酌。

  • 微软官方 下载 Windows 11
  • MSDN I Tell You
  • 新版 I Tell You

也可以单独下载镜像，使用第三方工具制作WinPE启动盘。推荐使用WinPE方式，因为不仅可以引导安装系统，还集成了丰富的维护工具。推荐使用纯净的PE制作工具，不要选择老毛桃、大白菜那种流氓工具。PE制作工具推荐如下：

  • 微PE工具箱 : 首推，简洁又纯净。
  • FirPE : 支持更换主题、ToDesk远程控制、浏览器、迅雷下载等功能。
  • Edgeless : 复杂支持很多自定义设置和安装的软件，但不支持老电脑。
  • HotPE : B站UP主VirtualHotBar开源工具。

原装ISO镜像制作工具，类似安装系统的光盘，插进去启动就行。现在很多笔记本和电脑放弃了光盘驱动器，所以开始使用U盘制作。这种方式简单直接，缺点是无法像PE那样选择系统列表，当无法进入系统时没有PE的维护工具可以使用。

  • 微软官方的制作工具。
  • Rufus : 开源纯净，软件非常小，无安装，支持Windows和Linux，支持Bios和UEFI。
  • 软碟通 UltraISO : 传统。
